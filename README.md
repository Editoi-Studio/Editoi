## Install dependences

```
Yarn start
```

## Build Windows 

```
Yarn dist
```
    
## Build Linux

Change package.json script dist options
    
```
 "dist": "build --tasks linux-x64 --mirror https://dl.nwjs.io/ ."
```

## Build Mac

Change package.json script dist options
    
```
 "dist": "build --tasks mac-x64 --mirror https://dl.nwjs.io/ ."
```
